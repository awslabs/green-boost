# Contributor's Introduction

Hello! Thank you for your interest in contributing to Green Boost. By contributing, you have the opportunity to scale your influence to AWS customers around the world!

To get started contributing:
1. Fork the repository by clicking [here](https://github.com/awslabs/green-boost/fork)
2. Clone your forked repository like: `git clone https://github.com/bestickley/green-boost.git`
3. Change directory: `cd green-boost`
4. Install dependencies: `pnpm i`
5. Try running linting: `pnpm lint`, typechecking: `pnpm typecheck` or tests: `pnpm test`.
