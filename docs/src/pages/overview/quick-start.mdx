# Quick Start

1. Follow the [prerequisites](./prereqs).
2. Install the Green Boost CLI: `pnpm add -g gboost`.
3. Create a web app with: `gboost create`.
4. Follow the prompts to select a template, directory, app id, and app title. To learn more about `gboost create` and the templates available, learn more [here](../cli/create).
5. Change directory into your directory: `cd <your-directory>`
6. Install dependencies: `pnpm i`.
7. Change directory into the infrastructure folder: `cd infra`
8. Deploy: `pnpm deploy:local`
9. View your web app the CloudFront URL printed in your terminal towards the end of output from the previous command.
10. Clean up: `pnpm destroy:local`.
