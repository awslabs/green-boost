# Packages

Green Boost is published into 5 packages on NPM:
- [gboost](https://www.npmjs.com/package/gboost): Command Line Interface (CLI)
- [gboost-common](https://www.npmjs.com/package/gboost-common): Utility library to share commonly needed code between other libraries.
- [gboost-infra](https://www.npmjs.com/package/gboost-infra): AWS CDK Constructs Library.
- [gboost-node](https://www.npmjs.com/package/gboost-node): Node.js backend code typically including functions run in AWS Lambda.
- [gboost-ui](https://www.npmjs.com/package/gboost-ui): React UI Component Library built on Material UI and Next.js.

Each package is versioned and released differently according to semantic versioning. Packages are only published as ES modules.
