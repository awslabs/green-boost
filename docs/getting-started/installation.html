Installation

TODO: add installation links to requirements

Requirements:

  • Git
  • PNPM
  • AWS CDK Toolkit
  • AWS Account
  1. Setup environment
pnpm env use --global 16
pnpm add -g gboost
note

Commitment issues? No problem. use: pnpm dlx gboost ... wherever you see gboost

  1. Scaffold your web app
gboost create # follow prompts
cd my-gb-app
pnpm i # install dependencies
  1. Boostrap your AWS account and deploy infrastructure
cdk bootstrap aws://123456789012/us-east-1
gboost deploy-dev
# remember to acknowledge IAM changes when prompted
  1. Visit the CloudFront URL logged by gboost deploy-dev
